VoxSherpa is a free, fully offline Text-to-Speech app powered by 
two professional-grade neural TTS engines.

🎙️ KOKORO ENGINE
Studio-quality voice synthesis using the Kokoro-82M neural model — 
the same technology that rivals ElevenLabs quality. Supports Hindi, 
English, and 50+ languages through the multi-language model.

⚡ PIPER ENGINE  
Fast, lightweight neural TTS using Piper VITS — generates natural 
speech in seconds. Ideal for everyday use.

✨ KEY FEATURES
• 100% offline — no internet, no cloud, no data sent anywhere
• Multiple voice models — download and switch anytime
• Save audio — export generated speech as WAV files
• Speech library — save and replay your generations
• Adjustable speed — control playback rate
• Smart generation progress — real-time sentence counter

🔒 PRIVACY FIRST
No ads. No tracking. No account required. Everything runs locally 
on your device.

Built with Sherpa-ONNX — the same inference engine used by 
professional offline AI applications.
